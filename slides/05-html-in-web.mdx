### Tag penting untuk pengunaan html di web

+ <a>&lt;!DOCTYPE html&gt;</a> // deklarasi type dokument
+ <a>&lt;html&gt; &lt;/html&gt;</a> // tag html
+ <a>&lt;head&gt; &lt;/head&gt;</a> // head website
+ <a>&lt;title&gt; &lt;/title&gt;</a> // title website
+ <a>&lt;body&gt; &lt;/body&gt;</a> // body website
